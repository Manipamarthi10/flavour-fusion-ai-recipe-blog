# Flavour Fusion: AI-Driven Recipe Blogging

Flavour Fusion is a cloud-based web application that uses Google Generative AI to create high-quality recipe blogs automatically.

## Features
- AI-generated recipe blogs
- Custom topic and word count
- Programmer joke during generation
- Clean and user-friendly UI

## Tech Stack
- Python
- Streamlit
- Google Gemini (Generative AI)
- NLP

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Add your API key in .env file
3. Run the app:
   streamlit run app.py

## Use Cases
- Food bloggers
- Home cooks
- Working professionals
