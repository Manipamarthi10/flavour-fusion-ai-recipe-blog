# Flavour Fusion: AI-Driven Recipe Blogging

Flavour Fusion is a cloud-based web application that leverages Google Generative AI to generate high-quality recipe blogs automatically.

## Features
- AI-powered recipe blog generation
- Custom recipe topic and word count
- Programmer joke during content generation
- Simple and user-friendly interface

## Technology Stack
- Python
- Streamlit
- Google Gemini (Generative AI)
- Natural Language Processing (NLP)

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Add your API key in a .env file
3. Run the application:
   streamlit run app.py

## Use Cases
- Food bloggers
- Home cooks
- Working professionals
